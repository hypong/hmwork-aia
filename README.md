# how to start

### install dependencies
```
npm i
```

### dev mode
```
npm run dev
```
### production mode
```
mpm run start
```

### production bulid
```
mpm run build
```

### eslint
```
npm run lint-fix
``` 

### jest
```
npm run test
``` 

### Limitation
* Because of my setting the Firebase Auth can only work on localhost
